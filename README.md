# 🌐 Personal Portfolio Website

A modern personal portfolio website built using **HTML, CSS, and JavaScript**. This project showcases my skills, projects, education, and contact information in a clean and interactive design.

## 🚀 Features

- Responsive design for all devices
- Animated typing effect
- Smooth scrolling navigation
- About Me section
- Skills section
- Projects section
- Contact section
- Social media links
- Clean and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure


portfolio/
│── index.html
│── style.css
│── script.js
└── README.md

## 📸 Preview

Open index.html in your browser to view the website.

## 💻 How to Run

1. Clone the repository

bash

git clone https://github.com/prathamesh-pdea/personal-portfolio.git

2. Open the project folder

bash

cd personal-portfolio

3. Open index.html in your browser.
